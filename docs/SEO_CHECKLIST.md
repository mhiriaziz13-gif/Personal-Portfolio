# SEO Checklist

## Included

- Route-specific title and meta description metadata.
- Canonical URL support through `NEXT_PUBLIC_SITE_URL`.
- Open Graph metadata and share image.
- `favicon.svg`.
- `app/sitemap.ts` generating `sitemap.xml`.
- `app/robots.ts` generating `robots.txt`.
- JSON-LD Person schema with name, email, locality, role focus and LinkedIn.
- Clean page and case-study URL structure.

## Before launch

1. Set `NEXT_PUBLIC_SITE_URL` to the verified live domain.
2. Confirm `https://your-domain/sitemap.xml` and `/robots.txt` resolve.
3. Test a shared Home URL in LinkedIn and a social-card debugger after launch.
4. Register the site in Google Search Console and submit the sitemap.
5. Do not add unverified employers, quantified outcomes or credentials solely for keyword coverage.
6. Update page descriptions if roles, project names or availability change.
